# Recuperacion Laravel - Blog

Aplicacion Laravel 12 con parte web Blade y API REST para posts y comentarios.

## Instalacion

1. Ejecutar composer install.
2. Copiar .env.example a .env.
3. Ejecutar php artisan key:generate.
4. Ejecutar php artisan migrate:fresh --seed.
5. Ejecutar php artisan serve.

Usuario de prueba: admin@example.com / password.
